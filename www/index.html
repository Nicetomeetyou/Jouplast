INSTALLATION SUPPORT CAILLEBOTIS
Fixez le support caillebotis sur un plot dalle.
Etape suivante
INSTALLATION SUPPORT CAILLEBOTIS
Placez le caillebotis sur deux plots, en l’insérant à la moitié du support de manière à pouvoir poser un autre caillebotis.
Etape suivante
INSTALLATION SUPPORT CAILLEBOTIS
Placez les autres caillebotis et posez le reste de votre terrasse normalement.
Retour aux conseils
INSTALLATION PLOTS LAMBOURDES
Posez une lambourde sur deux plots, de manière à ce qu’elle soit perpendiculaire au mur.
Etape suivante
INSTALLATION PLOTS LAMBOURDES
Réglez la hauteur des plots à l’aide de l’écrou et vérifiez que la lambourde soit à niveau à l’aide d’un niveau à bulle.
Etape suivante
INSTALLATION PLOTS LAMBOURDES
Placez les autres lambourdes et assurez-vous que l’ensemble de la structure soit à niveau.
Etape suivante
INSTALLATION PLOTS LAMBOURDES
Fixez les lames sur les lambourdes.
Retour aux conseils
INSTALLATION PLOTS DALLES

Etape suivante
INSTALLATION PLOTS DALLES
Réglez la hauteur des plots à l’aide de l’écrou.
Etape suivante
INSTALLATION PLOTS DALLES
Vérifiez que la dalle soit à niveau à l’aide d’un niveau à bulle.
